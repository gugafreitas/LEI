Detailed Description

Module : Guião1 + Guião2 Description : Parser -> (Guião1 e Guião2) - Trabalho de LI2 2020/2021

Este ficheiro tem como objetivo criar programas capazes de ler inputs e devolver outputs adequados às operações implementadas no próprio programa e assim relacioná-las com a stack criada noutro ficheiro.
